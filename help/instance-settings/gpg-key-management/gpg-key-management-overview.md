# Gestione delle chiavi GPG - Panoramica

GPG (GnuPG) crittografa i messaggi utilizzando coppie di chiavi singole asimmetriche. Per implementare la crittografia GPG con [!DNL Campaign], un utente amministratore deve installare e/o generare le chiavi GPG in un’istanza di marketing direttamente dal Pannello di controllo.
